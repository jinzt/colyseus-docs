# Colyseus文档

为[Colyseus](https://github.com/colyseus/colyseus/)使用编写的文档。

该项目使用Markdown编写使用[mkdocs](http://www.mkdocs.org)编译的文档。

## 安装和设置

```shell
pip install -r requirements.txt
```

## 运行

```
mkdocs serve
```

## 构建
```
mkdocs build
```
